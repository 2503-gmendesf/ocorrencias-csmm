# Sistema de Ocorrências — Santa Maria Minas

Sistema web responsivo para gerenciamento e análise de ocorrências escolares.

## Funcionalidades

- Login por perfil
- Registro de ocorrências
- Dashboard analítico
- Filtros por turma, professor e unidade
- Gráficos com Chart.js
- Interface responsiva

## Tecnologias

- HTML5
- CSS3
- JavaScript Vanilla
- Chart.js
- Tabler Icons

## Como executar localmente

Clone o repositório:

```bash
git clone https://github.com/SEU-USUARIO/smm-ocorrencias.git
```

Entre na pasta:

```bash
cd smm-ocorrencias
```

Abra o arquivo:

```bash
index.html
```

ou utilize a extensão Live Server do VSCode.
